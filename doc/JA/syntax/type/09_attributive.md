# 属性型

属性型は、レコードおよびデータクラス、パッチ、モジュールなどが含まれる型です。
属性型に属する型は値型ではありません。

## レコード型の合成

合成されたレコード型は平坦化できます。例えば、`{*{.name = Str; .age = Nat}; *{.name = Str; .id = Nat}}`は`{.name = Str; .age = Nat; .id = Nat}`となります。
