在编译时知道长度的可变长度数组。`[T; N]!`也有语法糖。

## methods

* push! ref! self(N ~> N+1, ...), elem: T

* pop! ref! (N ~> N-1, ...) -> T

* sample!(ref! self) -> T
* sample! ref! self, M: Nat -> [T; M]

  随机选择里面的元素并返回副本

* shuffle!(ref! self)

  打乱内容。

* assert_len ref! self(_ ~> N, ...), N: Nat -> () or Panic

  验证长度。
  长度不正确时会导致`panic`

## Impl

* From Range Int
* From [T; N]
* Num
